# Dataset Folder

This folder contains the datasets used for the B2B EdTech Lead Analytics Project.

## Files Included

### 1. b2b_edtech_leads.csv
Raw dataset containing lead generation, engagement, qualification, and conversion information collected from multiple marketing and acquisition channels.

### 2. clean_b2b_edtech_leads.csv
Cleaned and transformed dataset prepared for analysis and dashboard development.

## Data Preparation

The dataset was cleaned using Excel and Python by:

- Handling missing values
- Standardizing categorical fields
- Removing inconsistencies
- Creating analysis-ready data structures
- Preparing data for SQL queries and Power BI dashboards

## Purpose

The dataset was used to analyze:

- Lead Generation Performance
- Marketing Source Effectiveness
- Lead Quality & Engagement
- Customer Profiles
- Conversion Performance
- Enrollment Goals & Acquisition Trends
