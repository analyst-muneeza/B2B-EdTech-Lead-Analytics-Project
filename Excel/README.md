# Excel

This folder contains the Excel workbook used during the initial stages of the B2B EdTech Lead Analytics Project.

## File Included

### B2B_EdTech_Lead_Analytics.xlsx

The workbook was used for:

- Initial data exploration
- Data validation and quality checks
- Handling missing values
- Removing duplicate records
- Standardizing data formats
- Preparing the dataset for Python, SQL, and Power BI analysis

## Key Activities

- Data Cleaning
- Data Transformation
- Data Validation
- Basic Data Analysis

## Outcome

The cleaned dataset was exported for further analysis in Python, SQL, and Power BI to build an end-to-end analytics solution.
