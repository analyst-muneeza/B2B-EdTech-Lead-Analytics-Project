# Power BI

This folder contains the Power BI dashboard developed for the B2B EdTech Lead Analytics Project.

## File Included

### B2B_EdTech_Lead_Analytics_Dashboard.pbix

Interactive Power BI dashboard built to analyze lead generation, engagement, acquisition channels, customer profiles, and conversion performance.

## Dashboard Pages

### 1. Executive Overview
- Total Leads
- Converted Leads
- Conversion Rate
- Top Marketing Source
- Lead Conversion Funnel
- Engagement Level Distribution
- Lead Acquisition Channel Analysis

### 2. Marketing Performance Analysis
- Leads by Marketing Source & Lead Type
- Lead Quality Distribution
- Conversion Rate by Marketing Source
- High-Intent Leads by Lead Profile Type

### 3. Lead Engagement Analysis
- Website Visits by Engagement Level
- Conversion Rate by Lead Quality Level
- Lead Interaction Status Distribution
- Course Interest Area Analysis

### 4. Customer & Acquisition Analysis
- Customer Profile Distribution
- Enrollment Goal Analysis
- Lead Distribution by City
- Lead Profile Type Breakdown

### 5. Business Insights & Action Plan
- Key Business Insights
- Strategic Recommendations
- Actionable Growth Opportunities

## Features

- Interactive Slicers
- Drill-through Navigation
- KPI Cards
- Dynamic Visualizations
- Business Insights Dashboard
