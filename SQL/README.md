# SQL

This folder contains the SQL analysis performed for the B2B EdTech Lead Analytics Project.

## Files Included

### B2B_EdTech_Lead_Analytics.sql
SQL queries used to analyze lead generation, lead quality, customer engagement, marketing performance, and conversion trends.

### query_results.png
Screenshot of SQL query outputs and business insights generated from the database.

### supabase_table.png
Database table structure used for analysis in Supabase/PostgreSQL.

---

## Analysis Performed

### Lead Performance Analysis
- Total Leads
- Converted Leads
- Conversion Rate
- High-Intent Leads

### Marketing Source Analysis
- Top Marketing Sources
- Lead Acquisition Channels
- Source-wise Lead Distribution
- Marketing Performance Comparison

### Lead Quality Analysis
- Lead Quality Distribution
- High-Relevance Lead Analysis
- Conversion by Lead Quality Level

### Customer Analysis
- Customer Profile Distribution
- Enrollment Goal Analysis
- Geographic Lead Distribution
- Lead Profile Analysis

### Engagement Analysis
- Website Visit Analysis
- Engagement Level Analysis
- Lead Interaction Insights

---

## Business Insights Generated

- Identification of top-performing marketing channels
- Evaluation of lead quality and conversion effectiveness
- Understanding customer enrollment motivations
- Analysis of engagement behavior across lead segments
- Geographic distribution of leads and customer profiles
